# Contributing to SMS2Email

Thank you for your interest in contributing to SMS2Email! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/sms2email.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `npm install`
5. Copy `.env.example` to `.env` and fill in your credentials

## Development

### Run in Development Mode

npm run dev
